🥷🏾 Nairobi Ninja
Nairobi Ninja is a single-player 2D platformer built with Unity.
Players explore Kenyan-inspired levels filled with enemies, obstacles, and challenges in an adventurous side-scrolling experience.

🎮 Game Overview

Nairobi Ninja takes players through action-packed levels where they:

🏃 Move through 2D side-scrolling environments

🕷️ Encounter multiple enemy types

🪙 Collect items and score points

⚔️ Overcome platforming challenges

The game is designed to run on any personal computer using Unity.

🛠 Built With

Unity 2017.2.0f3

C#

Unity 2D Engine

⚠️ The project was originally developed using Unity 2017.2.0f3.
Using this version (or a close 2017.x version) is recommended to avoid compatibility issues.

🚀 How to Run the Project
1️⃣ Install Unity

Download and install:

Unity 2017.2.0f3

You can install this version through Unity Hub under the “Installs” section.

2️⃣ Clone or Download the Repository

Option A – Clone

git clone https://github.com/your-username/Nairobi-Ninja.git

Option B – Download ZIP

Click Code

Select Download ZIP

Extract the folder

3️⃣ Open the Project in Unity

Open Unity Hub

Click Add

Select the root Nairobi-Ninja folder

Open the project

Allow Unity to import and compile the scripts

4️⃣ Run the Game

Navigate to:

Assets → Scenes

Open MainMenu.unity or Gameplay.unity

Press ▶ Play in the Unity editor

🏗 Build a Standalone Game (Optional)

To export the game:

Go to File → Build Settings

Add both scenes to Scenes in Build

Select your platform (Windows, Mac, Linux)

Click Build

Choose a destination folder

Unity will generate an executable file.

⚠️ Troubleshooting

If you encounter issues:

Ensure you are using Unity 2017.2.0f3

If using a newer Unity version, allow Unity to upgrade the project when prompted

Make sure the folder structure remains intact (Assets, ProjectSettings, Packages must stay in the root)
